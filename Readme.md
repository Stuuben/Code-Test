# Code-Test

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Stuuben/Code-Test.git
   ```
2. Install NPM
   ```sh
   npm install
   ```
3. To start the project
   ```sh
   npm run dev
   ```
4. To run tests with jest
   ```sh
   npm test
   ```
5. create a .env file in the root of the project and insert your key.
   ```sh
   VITE_REACT_APP_ACCESS_KEY = <YOUR KEY>
   ```
